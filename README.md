# 课表成绩管理系统
## 一、项目简介
本项目是北京信息科技大学开源软件实践3组开发的校园工具，包含课表解析、成绩绩点计算、CI自动化三大核心模块，可实现课表文件读取、学分绩点自动核算、自动化代码校验。

## 二、运行环境
Python 3.8 及以上版本，仅使用Python标准库，无需额外安装第三方依赖。

## 三、安装步骤
1. 克隆项目仓库
```bash
git clone https://github.com/BISTU-OSSD/practice-group3.git
cd practice-group3
```

## 四、功能使用说明
1. 课表解析模块
运行命令：`python schedule.py`
读取Excel课表文件，自动提取课程、上课时间、授课教师信息，生成结构化课表数据。

2. 成绩计算模块
运行命令：`python calculator.py`
录入各科分数与对应学分，自动计算总学分、平均绩点，支持挂科、满分等特殊分数计算。

3. CI自动化模块
运行命令：`python ci_test.py`
批量执行全部单元测试，自动检测代码运行异常。

## 五、单元测试使用指南
项目所有测试用例统一存放在 `tests/` 文件夹内，一键执行全部测试脚本：

```bash
python -m unittest discover tests
```
